# Лендинг для Карате Федерации Московской области

*   **Тема проекта:** Сайт-визитка для региональной федерации карате.
*   **Цель:** Информирование о направлениях, наборе в группы и контактах.
*   **Использованный фреймворк:** [Bulma](https://bulma.io/) v1.0.2
*   **Дополнительно:** Font Awesome для иконок.
*   **Ссылка на Pages:** [https://artamenos.github.io/FMO_karate/](https://artamenos.github.io/FMO_karate/)
